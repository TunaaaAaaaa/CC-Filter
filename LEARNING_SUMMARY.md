## 🎓 技术知识点总结

### 1. 数据预处理技术

- **文件合并**: 高效合并多个数据文件

### 2. 文本清洗技术
- **启发式规则**: 基于经验的质量过滤规则
- **符号密度**: `symbols / total_length`
- **词长分析**: 平均词长、最短/最长词
- **重复检测**: 字符频率分析

### 3. 去重算法
**MinHash 原理**:
```
1. 将文本分词
2. 对每个词计算哈希值
3. 使用 k 个哈希函数生成签名
4. 比较签名的相似度
```

**LSH (局部敏感哈希)**:
```
1. 将签名分桶
2. 相似文档落入相同桶的概率高
3. 只比较同桶内的文档
4. 复杂度从 O(n²) 降到 O(n)
```

### 4. 语言识别技术
- **统计方法**: 基于n-gram统计
- **多语言支持**: 识别 50+ 种语言
- **置信度评估**: 提供识别可靠性指标

### 5. 质量评估技术
**启发式评分要素**:
```
- 文本长度 (越长约好)
- 词数统计 (词汇丰富度)
- 平均词长 (3-15字符最佳)
- 符号比例 (过高表示噪声)
- 字符重复 (重复内容检测)
```

**KenLM 统计模型**:
- **困惑度**: 衡量文本质量
- **n-gram 概率**: 基于统计的语言模型
- **归一化评分**: 长度无关的质量评估
